# Система выбора комплекта одежды

Веб-приложение для работы с онтологией одежды, позволяющее подбирать подходящие комплекты одежды в зависимости от сезона, типа осадков и цели выхода.

## Структура проекта

- `app.py` - Flask-приложение с SPARQL-запросами к онтологии
- `ChoiceClothes.rdf` - онтология одежды в формате OWL/RDF
- `templates/` - HTML-шаблоны для веб-интерфейса
- `requirements.txt` - зависимости Python

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Функциональность

- **Список элементов одежды** - просмотр всех элементов одежды с брендами и типами
- **Список сезонов** - просмотр доступных сезонов
- **Типы осадков** - просмотр типов осадков
- **Цели выхода** - просмотр целей выхода из дома
- **Одежда с характеристиками** - полный список одежды со всеми характеристиками
- **Подбор комплекта одежды** - поиск подходящей одежды по выбранным критериям (сезон, осадки, цель)

## Технологии

- Flask - веб-фреймворк
- RDFLib - работа с RDF/OWL онтологиями
- SPARQL - язык запросов к семантическим данным
