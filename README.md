# ChoiceClothes - Рекомендательная система одежды на основе онтологии

ChoiceClothes - это веб-приложение на Flask, которое использует онтологию одежды для предоставления рекомендаций по подбору одежды. Система анализирует предпочтения пользователя, текущие условия (погода, сезон, мероприятие) и предлагает соответствующие варианты одежды.
